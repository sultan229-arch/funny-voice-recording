# Gravadora de veu divertida

[View full project on microbit\.org](https://microbit.org/ca/projects/make-it-code-it/funny-voice-recorder)

To code this project yourself, right click on the background of the workspace and choose ‘delete all blocks’, then follow the coding video below. You can find the `forever` and `on start` blocks in the `Basic` section.

### En què consisteix?

Grava la teva veu amb el micròfon de la BBC micro:bit i reprodueix-la amb més velocitat o més lent.

### Instruccions

Posa el programa a continuació en una micro:bit. Prem el botó A i parla pel micròfon. Apareix un quadrat a la pantalla LED mentre s'està gravant. 

Prem el botó B per reproduir el so gravat. Es reprodueix el doble de ràpid, fent que la teva veu s'acceleri i soni grinyolant!

### Com funciona

El programa estableix la freqüència de mostreig a 10.000 Hertz (Hz) per a la gravació. Això significa que les mesures de la micro:bit, o mostres, sonen del micròfon 10.000 vegades cada segon.

Quan el reprodueix, reprodueix les mostres el doble de ràpid, 20.000 vegades cada segon. Això significa que reprodueix el doble de ràpid i duplica el to de tots els sons que hagi gravat.

Enregistrar un so nou suprimirà l'enregistrament anterior, igual que prémer el botó de restabliment de la part posterior o desconnectar el micro:bit de la seva font d'alimentació (USB o bateria).
